# API Endpoints for Test and Production Environments

## Trade Compliance Management Test Environments

The following endpoints are available for testing **Trade Compliance Management** integrations:

- `https://rz3.aeb.de/test4ce/`
- `https://rz3.aeb.de/test5ce/`

## Trade Compliance Management Production Environments

## Customer-Specific Test Environments

For customer-specific testing environments, use the following pattern:

- `https://rz3.aeb.de/{customer}test1ce/`

Replace `{customer}` with your specific customer identifier.

## Customer-Specific Production Environments

For customer-specific production environments, use the following pattern:

- `https://rz3.aeb.de/{customer}prod1ce/`

Replace `{customer}` with your specific customer identifier.
